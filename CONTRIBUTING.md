# Contributing

Thanks for your interest in contributing! This repository contains Gynecologist Assistant AI — a domain-specific chatbot built with Google Gemini and Chainlit.

## Getting set up

pip install -r requirements.txt; set GOOGLE_API_KEY env var; chainlit run app.py

## How to contribute

- **Bugs / ideas** — open an issue first using the templates so we can discuss the change.
- **Pull requests** — fork, branch from `main`, keep changes focused, and fill in the PR template.

## Guidelines

- Follow the existing code style and module layout.
- Never commit credentials, API keys, tokens, or personal data.
- Update the README if your change affects usage or setup.
- Test your changes before submitting.

## Commit messages

Short and imperative, e.g. `Fix null handling in transform step`.

By contributing, you agree that your contributions will be licensed under the repository's GPL-3.0 license.
